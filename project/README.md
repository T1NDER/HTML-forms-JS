# Popover Component

[![Build Status](https://github.com/t1nder/HTML-forms-JS/actions/workflows/gh-pages.yml/badge.svg)](https://github.com/t1nder/HTML-forms-JS/actions/workflows/gh-pages.yml)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)](https://t1nder.github.io/HTML-forms-JS/)

Bootstrap-like popover implementation in vanilla JavaScript without jQuery.

## 📖 Description

This project implements a popover component similar to Bootstrap's popover, but built with pure JavaScript. The popover appears above the trigger button and is centered horizontally relative to it.

## 🌐 Live Demo

Check out the live demo: [https://t1nder.github.io/HTML-forms-JS/](https://t1nder.github.io/HTML-forms-JS/)

## 🚀 Features

- Pure JavaScript (no jQuery dependency)
- Responsive positioning
- Accessible (ARIA attributes)
- Keyboard navigation (Escape to close)
- Click outside to close
- Built with Webpack
- Automated tests with Jest and JSDOM
- CI/CD with GitHub Actions
