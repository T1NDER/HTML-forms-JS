# Popover Component
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)](https://t1nder.github.io/HTML-forms-JS/)

Bootstrap-like popover implementation in vanilla JavaScript without jQuery.

## Installation

```bash
npm install
```

## 🛠️ Development

```bash
npm run dev
```

## 📁 Build

```bash
npm run build
```
